## 项目简介
此项目用于练习C++的使用，但本质还是为竞赛而生。目标只有一个——**在最短的时间内`AC(Accepted)`!**——不论是**暴力出奇迹**，还是**玄学优化**，我们都将它们纳入代码的怀抱。

## 项目特点
- **能跑就行**：代码风格？不存在的，只要能通过测试点，一切都是最优解。
- **算法大杂烩**：暴力搜索、动态规划、线段树······总有一款能用。
- **紧急赛前救命**：快速生成代码模板，五分钟从零开始造火箭。
- **Bug即特性**：当Bug成为题目的隐藏解法时，不必多言，默默承受即可。
